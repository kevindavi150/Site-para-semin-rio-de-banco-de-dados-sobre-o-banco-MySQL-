Entenda Sobre MySQL

Domine bancos de dados e leve seus projetos para outro nível com MySQL.

Criadores

Michael "Monty" Widenius, David Axmark e Allan Larsson

Início do Projeto

O MySQL foi criado em 1994, na empresa TcX na Suécia.

Primeira versão

Foi desenvolvida para atender à demanda de aplicações web dinâmicas, utilizando linguagens C e C++ para garantir alta performance e portabilidade./p>

Motivo

Necessidade

Um banco de dados mais rápido que o mSQL, que não suportava índices adequadamente.
